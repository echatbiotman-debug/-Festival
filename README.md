# 🎵 Beat Mawazin 2026

Application web pour le festival Mawazin - Rythmes du Monde à Rabat.

## 📋 Description

Cette application permet aux festivaliers de:
- Découvrir la programmation complète du festival
- Créer un planning personnalisé
- Marquer les concerts vécus dans un passeport musical
- Filtrer les artistes par jour et par genre musical

## 🚀 Technologies utilisées

- React 18
- Vite
- Context API pour la gestion d'état
- LocalStorage pour la persistance des données
